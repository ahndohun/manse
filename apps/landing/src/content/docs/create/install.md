Manse Creator is distributed from the open Manse repository marketplace. It runs inside Codex and does not require an OpenAI API key, Manse account, or publishing token.

## Add the plugin to Codex

1. **Open Plugins in Codex.** Use the Codex app's plugin manager and choose to add a repository marketplace.
2. **Add the Manse repository.** Use `https://github.com/ahndohun/manse` as the marketplace source.
3. **Install Manse Creator.** Select **Manse Creator**, then begin with "Create a Manse game about..."

:::tip[Creator promise]
You should be able to describe, preview, validate, and publish a game without manually editing source code.
:::

## What the plugin brings

Manse Creator gives Codex the engine contract, deterministic checks, the publishing workflow, and reusable design patterns it needs to build a standalone motion game Site.
